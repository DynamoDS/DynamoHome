Tests
tests/CLAUDE.md

Scoped to the test folder. Documents the two isolated suites — Jest + React Testing Library for unit tests and Playwright for e2e — their file patterns, and the global mocks (Chrome WebView, CSS modules, images) that are auto-applied to every unit test. Hard rule: no page.locator() calls in test files — all selectors live in Page Object classes.

Jest Playwright mocks POM
Locales
src/locales/CLAUDE.md

Lists all 14 locale files (en, cs, de, es, fr, it, ja, ko, pl, pt-BR, ru, zh-Hans, zh-Hant, en-GB) and the single rule that governs them: en.json is the source of truth; every key added there must be mirrored in all other files before a PR can merge.

Components
src/components/CLAUDE.md

Decision table for where to place new components: module-specific folders for single-module use, Common/ for anything used by two or more modules, and src/components/ for app-level concerns. Enforces the CSS Module co-location rule — every component must have a matching .module.css file in the same folder.

Agents
Specialized autonomous subprocesses. Claude launches them to delegate focused work — each has a defined role, a curated tool set, and deep project context baked in.
frontend-agent
.claude/agents/frontend-agent.md

Senior React + TypeScript engineer persona. Owns everything inside src/components/, src/locales/, and src/localization/. Enforces functional components, explicit prop interfaces, CSS Modules, no hardcoded text, and the data-testid placement rules (testid belongs on the component's own root element, never on a parent wrapper). Also responsible for running unit tests after every change.

React TypeScript CSS Modules localization data-testid unit tests
testing-agent
.claude/agents/testing-agent.md

E2E testing engineer persona. Writes and maintains Playwright tests following the Page Object Model pattern. Knows all three pages (Recent, Samples, Learning), the sidebar, and how the app receives data from the Dynamo host. Always uses data-testid selectors and delegates data-testid additions to the frontend-agent when they're missing from the source.

Playwright POM e2e data-testid
build-agent
.claude/agents/build-agent.md

Build and tooling engineer persona. Handles changes to Webpack 5 config, TypeScript config, ESLint rules, Jest setup, and npm scripts. Knows the hard constraint that the output path dist/build/index.bundle.js is hardcoded in the Dynamo desktop app and must never change. Diagnoses bundle failures and CI pipeline issues.

Webpack TypeScript ESLint Jest CI
